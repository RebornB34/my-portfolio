# cybersec-dashboard

## Project Overview
The cybersec-dashboard is a web application designed for cybersecurity monitoring and analytics, built with Next.js, React, and TypeScript. It provides users with real-time insights into security metrics, helping organizations to stay ahead in the cyber threat landscape.

## Tech Stack
- **Frontend:** Next.js, React, TypeScript
- **Styled Components:** For styling the application
- **Chart.js:** For data visualization
- **Axios:** For API calls
- **Node.js:** Backend services (if applicable)

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/RebornB34/my-portfolio.git
   cd my-portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Available Scripts
- **`npm run dev`** - Starts the development server
- **`npm run build`** - Builds the app for production
- **`npm run start`** - Starts the production server
